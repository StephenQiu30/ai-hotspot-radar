# 04 Hotspot Plan

## 目标

实现热点入库、去重、列表、详情、筛选和排序。

## 范围

- 新建 `hotspots` 和 `ai_analyses` 表。
- 按 URL + 来源建立唯一约束。
- 提供热点列表与详情 API。
- 支持关键词、来源、重要性、时间范围筛选。

## 数据模型

- `hotspots`：标题、URL、来源、关键词、发布时间、抓取时间、状态。
- `ai_analyses`：热点 ID、真实性、相关性分数、理由、重要性、摘要。

## API

- `GET /api/hotspots`
- `GET /api/hotspots/{id}`

## 验收标准

- 同一 URL + 来源不会重复入库。
- 热点列表可分页。
- 热点详情包含来源链接和 AI 分析。
- 可以按关键词、来源、重要性、时间范围筛选。
- 可以按时间、相关性或重要性排序。

## 非目标

- 不做复杂聚类。
- 不做历史趋势图。
