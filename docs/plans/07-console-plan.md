# 07 Console Plan

## 目标

实现可用的管理控制台。

## 范围

- 关键词管理页。
- 热点列表页。
- 热点详情页。
- 来源和设置页。
- 任务记录和通知状态页。

## 页面

- `/keywords`
- `/hotspots`
- `/hotspots/[id]`
- `/sources`
- `/settings`
- `/runs`
- `/notifications`

## 验收标准

- 可以通过控制台管理关键词。
- 可以查看热点列表和详情。
- 可以筛选和排序热点。
- 可以手动触发热点检查。
- 可以查看任务执行状态和邮件通知状态。

## 非目标

- 不做登录权限。
- 不做多租户视图。
- 不做营销首页。
