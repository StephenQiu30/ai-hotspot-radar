# 04-成功指标

## 5. 成功指标

### 5.1 业务成功标准

- 用户可完成关键词创建、编辑、删除和启停
- 用户可手动触发一次热点检查并看到结果
- 系统至少能从两个来源返回热点候选
- 热点榜单中每条记录可追溯到原始来源
- 邮件配置存在时可发送通知，未配置时不影响主流程
- 控制台可支持热点浏览、详情查看、来源追踪、筛选和排序
- 报告 API 可生成日报/周报，且只聚合 `active` 热点

### 5.2 产品质量指标（MVP）

- 关键词监控可用性（增删改查与启停可用）
- 热点检查稳定性（单一来源失败不影响整体任务）
- 热点事件可追溯性（事件详情有来源链接）
- AI 分析可解释性（真实性、相关性、理由、重要性、摘要齐全）
- 控制台筛选可用性（关键词、来源、重要性、时间可筛选）
- 降级可用性（模型、邮件或可选来源未配置时主流程不崩溃）
